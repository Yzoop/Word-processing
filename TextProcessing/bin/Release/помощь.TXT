#Загрузка файла
Загрузка файла.
Чтобы загрузить файл (docx или txt) выполните следующие действия (#1 ЗАГРУЗКА):
1) Нажмите на кнопку загрузить файл
2) В открывшемся окне проводника выберите нужный вам файл
3) Если файл загрузился (соответствующее сообщение в истории загрузки файлов), то вы можете переходить в раздел 'Форматирование слов'

После успешной загрузки файла вы можете наблюдать некоторые полезние данные о загруженном файле. Для повторной загрузки повторите шаги из '#1 ЗАГРУЗКА', описанные в этом помощнике.

#Форматирование слов
Форматирование слов.
После успешной загрузки файла, вам будет доступен раздел "форматирование слов". В нем вы можете: А) заменять римские числа (X => 10) на арабские. Для этого вам потребуется нажать на кнопку "Заменить римские на арабские".
Для Расшифровки сокращений (т.д., т.е. и тому подобные) нажмите на кнопку 'Расшифровка сокращений'. В открывшемся окне нажмите на кнопку "Следующее слово". Для изменения сокращения на произвольное слово введите в разделе "новое слово" слово для замены, а затем нажмите ОК.
После работы с сокращениями, нажмите "Сохранить все".
Для работы с местоимениями нажмите на кнопку "Замена местоимений". Откроется то же окно, что и в "Расшифровке сокращений". Для изменения местоимения в предложении вам так же потребуется ввести новое слово в раздел "новое слово", затем ОК, а в конце работы "Сохранить все".
⚫ Кнопка "Убрать заглавные буквы, где не требуется" проверяет каждое слово в предложении на соответствие в базе слов с большой буквы. Если слово не найдено в базе, то оно переписывается с маленькой буквы (все остальные так же маленькие). Начало предложения учтено.
⚫ Кнопка "Убрать переносы (-), где не требуется" по аналогии с кнопкой "...заглавные буквы..." сравнивает каждое слово, в котором присутствует символ "-" и, если это слово не пишется через дефис, программа удаляет символ "-" в проверяемом слове.
Параметры сохранения:
	-DOCX. Выбор данной радио-кнопки означает, что все изменения будут сохранены в формате Word (.docx расширение). Так же при нажатии этой кнопки пользователю будет доступно выбрать из раздела "форматирование Docx" радио кнопку "Изображения и таблицы в отдельные документы". При нажатии данной кнопки все изображения/таблицы(отдельные файлы!) сохранятся в той же папке что и рабочий файл (пусть рабочий файл - FILE.docx) с названием (Отдельные файлы!)"изображения/таблицы из FILE.docx",а так же отдельно текст в файле FILE.docx.
	-TXT. Выбор TXT-радио кнопки означает, что в новый текстовый документ (.txt) будут сохранены все ТЕКСТОВЫЕ данные (только символы, без рисунков и т.п.)
⚫ Кнопка "Просмотреть текущий файл" открывает файл, который на данный момент выбран пользователем для сохранения/форматирования.

#База местоимений и родственных слов
База местоимений и родственных слов.
В данном разделе пользователь может добавить в подразделе "Местоимения" новое местоимение, введя новое местоимение в поле возле слова "Местоимение", а затем нажав кнопку "Добавить и сохранить". Удаление происходит похожим образом: введение удаляемого местоимения в разделе удаление записи местоимения, а затем нажатием кнопки "удаление местоимения".
В разделе родственные слова пользователь может найти все однокоренные слова (или слова, которые содержат подслово), введя подслово/корень в раздел "Часть слова", а затем нажав кнопку "просмотреть однокоренные". Если пользователь удовлетворен поиском, он может сохранить найденный список слов в файл. В файле однокоренные слова сохранены через запятую.
